# 影视详情页重构设计

## 概述

将影视详情页从「左侧边栏 + 右侧堆叠卡片」布局重构为「头部 + Tab 导航」布局，提升信息层级清晰度和操作效率。

## 页面整体结构

```
┌─────────────────────────────────────────────────────────────┐
│  ← 返回    影视标题                        [保存] [发布]    │
├─────────────────────────────────────────────────────────────┤
│  [概览]  [视频上传]  [转码任务]     ← shadcn Tabs           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                    Tab 内容区域                              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**页面头部**：
- 左：返回按钮 + 影视标题 + 类型/状态标签
- 右：保存按钮 + 发布/下架按钮

**Tab 导航**：
- 使用 shadcn/ui Tabs 组件，`variant="line"` 样式
- 组件级 Tab（单路由，状态控制切换）
- 默认显示「概览」Tab

## 概览 Tab

```
┌─────────────────────────────────────────────────────────────┐
│  统计卡片行 (grid 4列)                                       │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐        │
│  │ 上传状态  │ │ 转码状态  │ │ 播放量    │ │ 完播率    │        │
│  │ 3/10 集   │ │ 进行中    │ │ 1,234    │ │ 45%      │        │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘        │
├─────────────────────────────────────────────────────────────┤
│  基础信息 Card                                               │
│  ┌─────────────────────────────────────────────────────────┐│
│  │ 封面图 [编辑]  标题: xxx        [编辑]                   ││
│  │                类型: 电影     [编辑]                     ││
│  │                分类: 动作     [编辑]                     ││
│  │                标签: #科幻 #冒险                         ││
│  │                简介: xxxxxx   [编辑]                     ││
│  │                状态: 已发布   [下拉]                      ││
│  └─────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────┘
```

### 统计卡片

| 卡片 | 电影类型 | 电视剧类型 |
|------|---------|-----------|
| 上传状态 | 文件大小 (如 2.4GB) | 已上传集数/总集数 (如 3/10 集) |
| 转码状态 | 任务进度或状态文字 | 同左 |
| 播放量 | 总播放次数 | 总播放次数 |
| 完播率 | 平均完播百分比 | 平均完播百分比 |

### 基础信息 Card

**布局**：
- 左侧：封面图缩略图 (16:9, 约 200px 宽) + 更换按钮
- 右侧：字段列表，每行一个字段

**内联编辑交互**：
- 每个字段右侧有编辑按钮（铅笔图标）
- 点击编辑按钮 → 字段变为输入框/下拉框 + 保存/取消按钮
- 保存后调用 `useUpdateVideo` mutation
- 标签字段：Tag 组件展示，点击可添加/删除

**可编辑字段**：
- 标题 (Input)
- 类型 (Select: 电影/电视剧/动漫)
- 分类 (Select: 从分类列表选择)
- 标签 (TagInput 或 Badge + 添加/删除)
- 简介 (Textarea)
- 状态 (Select: 草稿/已发布/已归档)

## 文件结构

```

├── index.tsx              # 主页面，Tab 导航
├── OverviewTab.tsx        # 概览 Tab
├── UploadTab.tsx          # 视频上传 Tab
├── TranscodingTab.tsx     # 转码任务 Tab
└── components/
    ├── VideoHeader.tsx    # 页面头部
    ├── StatCard.tsx       # 统计卡片
    ├── InlineEditField.tsx # 内联编辑字段
    └── UploadedFileList.tsx # 已上传文件列表
```

## 实现要点

1. **移除左侧边栏**：删除现有 12 列布局中的 col-span-3 侧边栏
2. **Tab 状态管理**：使用 `useState` 管理 `activeTab`，默认 `'overview'`
3. **内联编辑**：创建 `InlineEditField` 通用组件，支持 text/select/textarea 类型
4. **统计卡片**：从 video 对象计算统计数据，电影和电视剧显示不同内容
5. **复用现有组件**：UploadDropzone、UploadQueue、TranscodingTable 直接复用
